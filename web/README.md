# prismdb-web

> prismdbのWebページを提供するNuxt.jsのアプリケーションです

## Environment

- docker
- docker-compose

または

- node(新しめのやつ)
- npm(新しめのやつ)

## Usage

```bash
# ローカルでサーバが立ち上がります(http://localhost)
$ docker-compose -f ../docker-compose.dev.yml up
```

または

```bash
# 依存パッケージのインストール
$ npm i

# ローカルでサーバが立ち上がります(http://localhost)
$ npm run dev
```
